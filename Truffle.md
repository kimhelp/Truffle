# Truffle

```
npm i truffle

```

npx truffle compile

```sh
-build : 솔리디티 코드 컴파일된 내용이 들어감.
-contracts : 솔리디티 코드를 넣는 공간
-migrations : deploy
-test : 배포된 애들을 실행시켜보는 공간
-truffle-config.js
```
